## Introduction

Based on the UPEI scholar module, Islandora Scholar is designed to implement a feature-rich Institutional Repository. Provides a PHP interface to the bibutils tools.

## Requirements

This module requires the following modules/libraries:

* [Islandora](https://github.com/islandora/islandora)
* [Tuque](https://github.com/islandora/tuque)
* [Bibutils](http://sourceforge.net/p/bibutils/home/Bibutils/)

## Installation

Install as usual, see [this](https://drupal.org/documentation/install/modules-themes/modules-7) for further information.Follow instructions above to install bibutils.
