SUMMARY
-------

Cite-Proc

Provides a PHP interface to the Cite-Proc library.

This module is meant to fully encapsulate the functionality of the citeproc-js
library but for use within Drupal. At the moment it does not fully support
citeproc-js.

Useful Links.

Cite-Proc Library Documentation:
http://gsl-nagoya-u.net/http/pub/citeproc-doc.html

Cite-Proc Demo Page:
http://gsl-nagoya-u.net/http/pub/citeproc-demo/demo.html

The citeproc-php library has been modified, do not just replace with the latest
 version.

class_rtf.php comes from the biblio drupal project, and has been modified for
use here.

REQUIREMENTS
------------

The following Drupal modules are required:
 * csl
