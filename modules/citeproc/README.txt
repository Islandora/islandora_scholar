## Introduction

Provides a Drupal interface to the citeproc-php library.

## Requirements

This module requires the following modules/libraries:

 * [csl](https://github.com/Islandora/citeproc-php)

## Installation

Install as usual, see [this](https://drupal.org/documentation/install/modules-themes/modules-7) for further information.

## Configuration

Cite-Proc Demo Page:
http://gsl-nagoya-u.net/http/pub/citeproc-demo/demo.html

The citeproc-php library has been modified, do not just replace with the latest
 version.

class_rtf.php comes from the biblio drupal project, and has been modified for
use here.
