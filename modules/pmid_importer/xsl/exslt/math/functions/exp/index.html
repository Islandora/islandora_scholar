Function Syntax

number math:exp(number)

The math:exp function returns e (the base of natural logarithms) raised to a power.

The return value is the power of the constant e. The constant e is Euler's constant, approximately equal to 2.178.

Implementations

The following XSLT processors support math:exp:

  • 4XSLT, from 4Suite. (version 0.12.0a3)
  • libxslt from Daniel Veillard et al. (version 1.0.19)

Implementations of math:exp are available in the following languages:

  • Msxsl (by James Fuller)
  • Javascript (by James Fuller)
