Function Syntax

string str:decode-uri(string, string)

The str:decode-uri function decodes a string that has been URI-encoded.

Implementations

There are currently no XSLT processors that we know of that support the latest version of str:decode-uri natively.

The following processors support earlier versions of str:decode-uri:

  • 4XSLT, from 4Suite. (version 0.12.0a3)

There are currently no third-party implementations of the latest version of str:decode-uri.

The following third-party implementations of earlier versions of str:decode-uri are available:

  • Javascript (by Chris Bayes)
  • Msxsl (by Chris Bayes)
