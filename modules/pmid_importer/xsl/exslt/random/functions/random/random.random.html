Function Syntax

number random:random()

The random:random function returns a random number from 0 to 1.

An implementation of this extension function in the EXSLT random namespace must conform to the behaviour described in this document.

Implementations

Built-in support for random:random is available in the following XSLT processors:

ProcessorProcessor VersionImplemented Version
4XSLT, from 4Suite.0.12.0a31

There are currently no third-party implementations of random:random.
