SUMMARY
-------

RIS Importer

Importer plugin for RIS files, to create citation objects.

REQUIREMENTS
------------

The following Drupal modules are required:
 * islandora_importer
 * islandora_scholar
 * bibutils

INSTALLATION
------------

Enable the module in the admin/modules page.
