SUMMARY
-------

Islandora Bibliography

Allows users to maintain/export their own bibliography of citations.

REQUIREMENTS
------------

The following Drupal modules are required:
 * islandora
 * csl
 * citeproc
 * islandora_bookmark

INSTALLATION
------------

Enable the module in the admin/modules page.
