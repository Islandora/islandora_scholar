SUMMARY
-------

EndnoteXML Importer

Allows users to import EndnoteXML files.

REQUIREMENTS
------------

The following Drupal modules are required:
 * islandora_importer
 * islandora_scholar
 * bibutils

INSTALLATION
------------

Enable the module in the admin/modules page.
